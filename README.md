# Kuma
---
### Inventory Management Software

Our management system is designed to efficiently oversee warehouses and their inventories, specifically catering to food ingredients and food-related products. However, its versatile functionality makes it suitable for managing any type of products you wish to use it for!

### Technologies Used
#### Backend:
  - Java Spring Boot
  - Postgresql
#### Frontend:
  - React

### Installation:

Clone the repository:

```sh
git clone https://github.com/d-swe/kuma.git
```

Set up backend:

```sh
cd backend
mvn install
mvn spring-boot:run
```

Set up frontend:

```sh
cd /kuma/frontend/kuma-ui
npm install
npm run dev
```
***Note: you will need to create your own local database *preferably* on PgAdmin and create mock data***
